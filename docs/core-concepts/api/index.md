# REST API

Hlavní způsob komunikace s Krameriem představuje REST API.

API poskytuje přístup k:

- digitálním objektům
- metadatům
- stránkám dokumentů
- vyhledávání
- správě procesů
- administrativním operacím

REST API je využíváno klientskými aplikacemi i integračními systémy.
