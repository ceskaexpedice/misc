# Akubra

igitální objekty jsou ukládány v repository vrstvě.

Repository zajišťuje:

- ukládání objektů
- správu datastreamů
- verzování dat
- přístup k digitálnímu obsahu

Implementace repository je založena na Fedora repository a modulu Akubra.
