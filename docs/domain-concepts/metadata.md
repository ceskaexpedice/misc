# Metadata

Každý objekt obsahuje metadata popisující jeho obsah.

Metadata mohou obsahovat například:

- název
- autora
- datum vydání
- jazyk
- typ dokumentu
- licenční informace

Metadata jsou ukládána v repository a současně indexována pro účely vyhledávání.
