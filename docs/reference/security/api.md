# Security API

Přehled endpointů souvisejících s autorizací a autentizací.

## Current User

```http
GET /api/user
```

Vrací informace o aktuálně přihlášeném uživateli.

## Permissions

```http
GET /api/permissions
```

Vrací efektivní oprávnění uživatele.

## Roles

```http
GET /api/roles
```

Vrací role dostupné v systému.
