# Actions

Akce představují oprávnění používaná autorizačním systémem.

## Available Actions

| Action | Description |
|----------|----------|
| VIEW_ITEM | Zobrazení objektu |
| READ_METADATA | Čtení metadat |
| WRITE_METADATA | Editace metadat |
| DELETE_OBJECT | Mazání objektů |
| ADMIN_ACCESS | Administrace systému |

## Action Usage

Každá chráněná operace deklaruje jednu nebo více požadovaných akcí.
