# 🚨 Řešení problémů (Troubleshooting)

Tato sekce pomáhá řešit problémy, které mohou nastat při práci se systémem Kramerius.

Obsahuje nejčastější chybové stavy a doporučené postupy jejich řešení.

## 📄 Dokument se nezobrazuje

➡️ [Řešení problému](document-not-visible)

## Jak postupovat

1. Zkontrolujte stav procesu v Admin klientovi
2. Otevřete detail procesu
3. Projděte logy
4. Postupujte podle konkrétního návodu

---

## Kdy použít tuto sekci

Použijte Troubleshooting, pokud:

- import selhal
- indexace neběží nebo skončila chybou
- dokument se nezobrazuje ve vyhledávání
- proces zůstává viset nebo se nezpracovává
