# Návody pro administrátora

Tato sekce obsahuje praktické návody pro správu a provoz systému Kramerius.

Administrátor zajišťuje instalaci, konfiguraci, zabezpečení a každodenní provoz jednotlivých komponent systému. Součástí jeho práce je také monitoring služeb, správa logů, aktualizace systému a řešení provozních problémů.
