# Admin Client – Návody pro administrátora

## Pro koho je tato stránka
Tato stránka je určena pro **administrátory**, kteří chtějí spravovat digitální knihovnu přes **Admin UI klienta**.  
Pokud hledáte pouze koncepty, vraťte se na [Getting Started – Administrátor](../getting-started/admin).

---

## 1. Přihlášení do Admin UI
- Přístup přes URL Admin UI klienta
- Přihlášení uživatele s rolí administrátor
- Kontrola platnosti tokenu a oprávnění

---

## 2. Správa sbírek a dokumentů
- Vytváření a úprava sbírek
- Import dokumentů přes procesní framework
- Hromadné operace (delete, update metadat)
- Kontrola stavu importů a procesů

> Detailní parametry a proměnné → [Reference – Admin Client](../reference/Admin_UI.md)

---

## 3. Správa uživatelů a rolí
- Přidání nového uživatele
- Přiřazení role (čtenář, kurátor, administrátor)
- Kontrola oprávnění a audit aktivit

> Viz také [Security Model](core-concepts/Security) a [Guides – Security](Security)

---

## 4. Nastavení notifikací a logů
- Zobrazení logů operací
- Konfigurace notifikací pro dokončené procesy
