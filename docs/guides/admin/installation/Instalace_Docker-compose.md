### Kramerius docker compose 

Pro snadnější manipulaci a instalaci vznikl následující projekt: [Kramerius docker compose](https://github.com/ceskaexpedice/kramerius-docker-compose?tab=readme-ov-file#docker-compose), který obsahuje všechny potřebné komponenty již kontejnerizované. 

Po dodržení instalačních kroků stačí spustit krameria příkazem: `docker compose up`
